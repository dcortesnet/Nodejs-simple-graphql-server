# Nodejs Graphql Apollo Server &middot; ![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)

Basic server with Nodejs and the addition of Graphql

## Install

```
npm install
```

## Run server

Server listen default on `http://localhost:4000/`

```
npm run dev
```
